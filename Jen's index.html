In plot():

    • o = default box
    • n = no box
    • l, 7, c, u, or ] = boxes that resemble their uppercase shapes
  2. las=x alters the axes label orientation:
    • 0 = default (parallel to axis)
    • 1 = horizontal
    • 2 = perpendicular
    • 3 = vertical
  3. cex=x, changes text font size inside plot; cex.main=x alters size of main title text, cex.lab=x changes axis label text size, and cex.axis=x alters the axis tick values text
    • 1 = default
    • 0.5 = half default size
    • 1.5 = 1.5 default size

DF

Degrees of freedom (DF) = n-1 if sampling distribution of sample means is normal.

The higher the DF, the more normal the distributions becomes (30 is close to normal).

Tests

One-Sample t-Test

Use for testing the mean of one quantitative variable.

“Test statistic measures how far the sample mean is from \(\mu\).” - Kruskal-Wallis Assessment Quiz

  • What’s the average height of women in the US?

When…

  • data is normally distributed
  • population mean (\(\mu\)) is known
  • sample represents population (SRS or large sample size (30+))

Testing Assumptions:

library(car)
qqPlot(NameOfYourData$Y)

Paired-Sample t-Test

Use for testing the mean of differences (pre-test/post-test and matched-pairs scenarios).

  • Are people are taller in the morning than the evening?

When…

  • data is normally distributed
  • population mean (\(\mu\)) is known
  • sample of differences represents population differences (SRS or \(n\) of the differences is large)

Testing Assumptions:

library(car)
qqPlot(Y1 - Y2)

Wilcoxon Signed-Rank

Use for testing the median of one quantitative variable or the difference of paired data. May also be used with ordinal data.

  • Are the people in my family taller in the morning than the evening?

When…

  • data is not normally distributed
  • the sample size is small
  • there are relatively few ‘ties’ in the paired data

With paired data, you might use a dotplot to check for ties in data (differences). They would show up as zeros on the plot.

stripchart(object ~ group, data=NameOfYourData, method="stack")

One-Sample t-Test

To run the t-test:

t.test(NameOfYourData$Y, mu = YourNull, alternative = YourAlternative, conf.level = 0.95)
  • where Y is numeric
  • alternative = “two.sided”, “greater”, “less”
  • conf.level = 1-\(\alpha\), usually 0.95

To test assumptions:

library(car)
qqPlot(NameOfYourData$Y)

Paired-Sample t-Test

To run the test:

t.test(NameOfYourData$Y1, NameOfYourData$Y2, paired = TRUE, mu = YourNull, alternative = YourAlternative, conf.level = 0.95)
  • where Y1 and Y2 are numeric
  • alternative = “two.sided”, “greater”, “less”
  • conf.level = 1-\(\alpha\), usually 0.95

To test assumptions:

library(car)
qqPlot(NameOfYourData$Y1 - NameOfYourData$Y2)

Independent Samples t-Test

Use for testing the difference between the means of two (possibly) different populations.

  • Are men are taller than women?

When…

  • data is normally distributed
  • population mean of the differences (\(\mu\)) is known
  • sample represents population (SRS or \(n\) of the differences is large))

Testing Assumptions:

library(car)
qqPlot(Y ~ X, data=YourData)

Example: See High School Seniors analysis for an example.

Wilcoxon Rank Sum (Mann-Whitney)

Use for testing equality of medians of 2 (possibly) different distributions of a quantitative variable. This test is like the Independent Samples t Test but is nonparametric. If testing 3+ distributions, use the Kruskal-Wallis Rank Sum test.

“Sums the ranks from one of the groups for its test statistic.” - Kruskal-Wallis Assessment Quiz

  • Are the men in my family taller than the women?

When…

  • data is not normally distributed
  • the sample size is small
  • there are relatively few ‘ties’ in the paired data

Use ‘stochastically greater’ when distributions are not identically distributed.

To check for ties use a boxplot overlayed with a stripchart. See Recalling Words analysis for an example or “code repository” for more info.

boxplot(y~x) 
stripchart(y~x)
    where y= numeric vector and x= is the factor or character vector

One-way ANOVA

Use for testing several means simultaneously. Uses 1 quantitative variable and 1 qualitative factor with multiple levels.

The ratio of the “between groups variation” to the “within groups variation” provides the test statistic for ANOVA. Note that the test statistic of ANOVA is an \(F\) statistic.

\[ F = \frac{\text{Between groups variation}}{\text{Within groups variation}} \]

  • Is the average height of women the same in all states?

When…

  • sample represents population (SRS or \(n\) of the differences is large))
  • data is normally distributed in each group (check residuals - Normal Q-Q)
  • population variance is the same for each group (check constant variance - Residuals vs Fitted)

Testing Assumptions:

par(mfrow=c(1,2))
plot(chick.aov, which=1:2, pch=16)
par(mfrow=c(1,1)) #resets par

Kruskal-Wallis Rank Sum

Use for testing if 3+ samples come from the same population or if at least one is from a different one. Ordinal data can be used. Use Wilcoxon Rank Sum if only testing 2 samples. Don’t reference spread or consistancy of data. Concentrate on shift!

H - “It is a combined measurement of how much the average ranks differ between each group.” - Kruskal-Wallis Assessment Quiz

  • Does a woman’s hair color affect her annual income?

When…

  • populations are assumed to be about the same form, just shifted or translated
  • data is not normally distributed

Testing Assumptions:

None

Notes:

  • Avoid min, max, + sd comparisons; focus on mean, median, 1Q + 3Qs instead
  • Graphs can be misleading, if p-value is high it may be due to random chance patterns
  • Use color for directing focus where desired

Math Model

One-way ANOVA
Hypotheses:

\[ H_0: \alpha_1 = \alpha_2 = \ldots = 0 \] \[ H_a: \alpha_i \neq 0 \ \text{for at least one} \ i \]

$$
H_0: \alpha_1 = \alpha_2 = \ldots = 0
$$
$$
H_a: \alpha_i \neq 0 \ \text{for at least one} \ i
$$
Math Model:

\[ Y_{ij} = \mu + \alpha_i + \epsilon_{ij} \] where \(\mu\) is the grand mean, \(\alpha_i\) is a factor with at least two levels, and \(\epsilon_{ijk} \sim N(0,\sigma^2)\) is the error term.

$$
Y_{ij} = \mu + \alpha_i + \epsilon_{ij}
$$
Two-way ANOVA
Hypotheses:

With a two-way ANOVA there are three sets of hypotheses.

One for the first factor: \[ H_0: \alpha_1 = \alpha_2 = \ldots = 0 \] \[ H_a: \alpha_i \neq 0 \ \text{for at least one} \ i \]

$$
H_0: \alpha_1 = \alpha_2 = \ldots = 0
$$
$$
H_a: \alpha_i \neq 0 \ \text{for at least one} \ i
$$

One for the second factor: \[ H_0: \beta_1 = \beta_2 = \ldots = 0 \] \[ H_a: \beta_j \neq 0 \ \text{for at least one} \ j \]

$$
H_0: \beta_1 = \beta_2 = \ldots = 0
$$
$$
H_a: \beta_j \neq 0 \ \text{for at least one} \ j
$$

One for the interaction of the two factors: \[ H_0: \text{The effect of} \ \alpha \ \text{is the same for all levels of} \ \beta \] \[ H_a: \text{The effect of} \ \alpha \ \text{is different for at least one level of} \ \beta \]

$$
H_0: \text{The effect of} \ \alpha \ \text{is the same for all levels of} \ \beta
$$
$$
H_a: \text{The effect of} \ \alpha \ \text{is different for at least one level of} \ \beta
$$
Math Model:

\[ Y_{ijk} = \mu + \alpha_i + \beta_j + \alpha\beta_{ij} + \epsilon_{ijk} \] where \(\mu\) is the grand mean, \(\alpha_i\) is one factor with at least two levels, \(\beta_j\) is another factor with at least two levels, \(\alpha\beta_{ij}\) is the interaction of the two factors, and \(\epsilon_{ijk} \sim N(0,\sigma^2)\) is the error term.

Note that more than two factors are allowed, i.e., more than \(\alpha_i\) and \(\beta_j\), and is sometimes called three-way ANOVA, four-way ANOVA, and so on. It is not required to include interaction terms, like \(\alpha\beta_{ij}\). The error term \(\epsilon\) is always required.

$$
Y_{ijk} = \mu + \alpha_i + \beta_j + \alpha\beta_{ij} + \epsilon_{ijk}
$$
Kruskal-Wallis Rank Sum
Hypotheses:

\[ H_0: \text{All samples are from the same distribution.} \] \[ H_a: \text{At least one sample's distribution is stochastically different.} \]

H_0: \text{All samples are from the same distribution.}
H_a: \text{At least one sample's distribution is stochastically different.}

Code Repository

One-way ANOVA
Run the test: 
myaov <- aov(y ~ A, data=YourDataSet)
summary(myaov)
Check assumptions: 
par(mfrow = c(1, 2))
plot(myaov, which=1:2)
par(mfrow = c(1, 1)) #to reset graphs
  • y must be a “numeric” vector of the quantitative response variable.
  • A is a qualitative variable (should have class(A) equal to factor or character. If it does not, use as.factor(A) inside the aov() command
Two-way ANOVA
Run the test: 
myaov <- aov(y ~ A+B+A:B, data=YourDataSet)
summary(myaov)
Check assumptions: 
par(mfrow = c(1, 2))
plot(myaov, which=1:2)
par(mfrow = c(1, 1)) #to reset graphs
  • y must be a “numeric” vector of the quantitative response variable.
  • A is a qualitative variable (should have class(A) equal to factor or character. If it does not, use as.factor(A) inside the aov() command.
  • B is a second qualitative variable that should also be either a factor or a character vector.
  • Note that factors C, D, and so on could also be added + to the model if desired.
  • A:B denotes the interaction of the factors A and B. It is not required and should only be included if the interaction term is of interest.
Kruskal-Wallis Rank Sum

kruskal.test(x ~ g, data=YourDataSet)
  • x is a numeric vector of data values that represents the quantatitive response variable.
  • g is a qualitative grouping variable defining which groups each value in x belongs to. It must either be a character vector or a factor vector.

For an alternative, see R Instructions under the Kruskal-Wallis Rank Sum tab.

Simple Linear Regression

Use for testing a single quantitative response variable (Y) and a single quantitative explanatory variable (X).

Test statistic = t

  • What affect does height have on shoe size?

Use when…

  • Linear Relation: the true regression relation between Y and X is linear.
  • Normal Errors: the error terms \(\epsilon_i\) are normally distributed with a mean of zero.
  • Constant Variance: the variance \(\sigma^2\) of the error terms is constant (the same) over all Xi values.
  • Fixed X: the Xi values can be considered fixed and measured without error.
  • Independent Errors: the error terms \(\epsilon_i\) are independent.

Testing assumptions (1, 2, 3, and 5):

The slope is interpreted as, “the change in the average y-value for a one unit change in the x-value.”

The y-intercept is interpreted as, “the average y-value when x is zero.” It is often not meaningful, but is sometimes useful.

Examples: Classmate’s, mine, Brother Cromar created this one.

Math Model

Simple Linear Regression shortcut
Hyoptheses:
Intercept hypotheses (sometimes useful):

\[ H_0: \beta_0 = 0 \\ H_0: \beta_0 \neq 0 \]

$$
H_0: \beta_0 = 0 \\
H_0: \beta_0 \neq 0
$$

If \(\beta_0 = 0\), then the model reduces to \(Y_i = \beta_1X + \epsilon_i\), a line going through the origin. This means the average \(Y\)-value is \(0\) when \(X = 0\) if the null hypothesis is true.

Slope hypotheses (most common):

\[ H_0: \beta_1 = 0 \\ H_a: \beta_1 \neq 0 \]

$$
H_0: \beta_1 = 0 \\
H_a: \beta_1 \neq 0
$$

If \(\beta_1 = 0\), then the model reduces to \(Y_i = \beta_0 + \epsilon_i\), which is a flat line. This means \(X\) does not improve our understanding of the mean of \(Y\) if the null hypothesis is true.

Math Models:
True regression model:

\[ \underbrace{Y_i}_\text{Some Label} = \overbrace{\beta_0}^\text{y-int} + \overbrace{\beta_1}^\text{slope} \underbrace{X_i}_\text{Some Label} + \epsilon_i \quad \text{where} \ \epsilon_i \sim N(0, \sigma^2) \]

$$
\underbrace{Y_i}_\text{Some Label} = \overbrace{\beta_0}^\text{y-int} + \overbrace{\beta_1}^\text{slope} \underbrace{X_i}_\text{Some Label} + \epsilon_i \quad \text{where} \ \epsilon_i \sim N(0, \sigma^2)
$$
Estimated regression line:

\[ \underbrace{\hat{Y}_i}_\text{Some Label} = \overbrace{b_0}^\text{est. y-int} + \overbrace{b_1}^\text{est. slope} \underbrace{X_i}_\text{Some Label} \]

$$
\underbrace{\hat{Y}_i}_\text{Some Label} = \overbrace{b_0}^\text{est. y-int} + \overbrace{b_1}^\text{est. slope} \underbrace{X_i}_\text{Some Label}
$$

Where:

  • lower case letters = samples, capitals = parameters (rarely known, ‘God’s law’)
  • \(y_i\) = individual y value
  • \(\bar{y}\) = average of all y-values in sample (it will always be a flat line, no x involved)
  • \(\hat{y}\) = a sloped line, x is figured in
  • \(\hat{y_i}\) = the single dot on \(\hat{y}\) line
  • residual = \(y_i - \hat{y_i}\)
  • \(\epsilon_i\) is the error term or the distance of the measured dot to the true line.
  • \(r_1\) is the residual or the distance of the measured dot to the estimated line.
  • \(\beta_0\) is the true y-intercept
  • \(b_0\) is the estimated y-intercept
  • \(\beta_1\) is the true slope
  • \(b_1\) is the estimated slope (the change in the average \(y_i\) value)
  • \(E\{Y_i\}\) is the expected value of \(Y_i\), the true line (the regression relation), usually unknown (dotted line)
  • \(\hat{Y_i}\) is the estimated (fitted) line, the line we get from a sample of data (solid line)
  • \(Y_i\) is the data, the regression relation plus an error term (the dots)

\[ H_0: \beta_1 = 0 \\ H_a: \beta_1 \neq 0 \]

If \(\beta_1 = 0\), then there is no relationship between \(x_i\) and the log of the odds that \(Y_i = 1\). In other words, \(x_i\) is not useful in predicting the probability that \(Y_i = 1\). If \(\beta_1 \neq 0\), then there is information in \(x_i\) that can be utilized to predict the probability that \(Y_i = 1\), i.e., the logistic regression is meaningful.

Math Model:

\[ P(Y_i = 1|x_i) = \frac{e^{\beta_0+\beta_1 x_i}}{1+e^{\beta_0 + \beta_1 x_i}} = \pi_i \]

Tests

Chi Squared Test of Independence

Compares observed counts with expected counts to see if the variables are associated (\(H_a\)) or independent (\(H_0\)).

Test Statistic: \(X^2\)

Requirements:

  • All expected counts should be \(\ge5\)

Will also work if all expected counts are \(\ge1\) and the average of expected counts is \(\ge5\)

Used when

  • there are only 2 variables and
  • both are categorical

Math Model

Hypotheses:

\[ H_0: \text{The row variable and column variable are independent.} \]

\[ H_a: \text{The row variable and column variable are associated (not independent).} \]

$$
H_0: \text{The row variable and column variable are independent.}
$$

$$
H_a: \text{The row variable and column variable are associated (not independent).}
$$
Math Model:

The following formula gives us the \(X^2\) test statistic:

\[ \chi^2 = \sum_{i=1}^m \frac{(O_i - E_i)^2}{E_i} \]

\chi^2 = \sum_{i=1}^m \frac{(O_i - E_i)^2}{E_i}
  • \(O_i\) represents each of the \(m\) observed counts
  • \(E_i\) represents the \(m\) expected counts
  • \(m = r\times c\)
    • where \(r\) represents the number of rows in the contingency table and
    • \(c\) represents the number of columns.

The expected counts are obtained by the formula

\[ E_i = \frac{\text{(row total)(column total)}}{\text{(total total)}} \]
